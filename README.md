# Rouge-like-Shooter

A top-down rogue-like shooter built in Python using Pygame, structured across multiple files.

## Features
- Enemy AI with pathfinding and animations
- Multiple weapon types with switching mechanics
- Bullet physics and collision detection
- Screen shake and fade transition effects
- Round-based progression system
- Between-round upgrade shop
- Dynamic lighting and overlay effects
- Particle effects and explosions
- Health, ammo and dash systems

## Built with
- Python
- Pygame
